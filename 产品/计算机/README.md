人们需要一台可改变功能（行为）的机器，造出来以后，起名字叫**computer** (计算机)
> 计算机把 **功能定义** 和 **实体机器** 分离，实体机器也叫**裸机**，作为功能定义的解析器。  
> 功能定义称作**程序**，写作程序叫**编程**

## 计算机网络 
> 满足机器间联网传输信息的功能

- ①计算机网络建立的主要目的是实现计算机资源的共享。计算机资源主要是指计算机硬件、软件与数据。
- ②互连的计算机是分布在不同的地理位置的多台独立的 “自治计算机”。连网的计算机既可以为本地用户提供服务，也可以为远程用户提供网络服务。
- ③连网计算机之间遵循共同的网络协议
